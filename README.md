## ğŸ“¦ Getting Started

### Prerequisites
*   Node.js (v14+)
*   npm or yarn

### Installation

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/Justinhxrsh/CICMS.git
    cd CICMS
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    ```

3.  **Build the project**:
    ```bash
    npm run build
    ```

4.  **Start the server**:
    ```bash
    npm start
    ```

The game will be available at `http://localhost:6500`.
